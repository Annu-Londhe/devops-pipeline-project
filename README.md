# 🚀 DevOps Pipeline Project (AWS + Kubernetes + Docker)

## 📌 Overview
This project demonstrates an end-to-end DevOps pipeline using Docker, Kubernetes, and CI/CD concepts.

## 🛠️ Tech Stack
- AWS (EKS, EC2)
- Docker
- Kubernetes
- Jenkins (CI/CD)
- ArgoCD (GitOps)

- Containerized application using Docker
- Kubernetes deployment with scaling
- Service exposure using NodePort
- GitOps-ready structure
